Html

Html staat voor Hypertext Markup Language. Dit is een codetaal die gebruikt wordt om structuur en inhoud te geven aan jouw webpagina.
Doormiddel van deze taal kan de browser lezen waarover de desbetreffende pagina gaat.
Dit is essentieel bij een webpagina, omdat hierdoor de structuur en inhoudt leesbaar is.




Css

CSS is een aanvulling op HTML, het staat voor Cascading Style Sheets oftewel CSS.
Zoals de naam al zegt zijn dit de zogenaamde “stylesheets” het representeert de opmaak van de pagina.
CSS is dan ook een aanvulling op de HTML-doormiddel hiervan kan er opmaak aan de pagina worden gegeven.
Ook kunnen animaties en dergelijke elementen met CSS kunnen worden gerealiseerd.
Zo kan bijvoorbeeld een button animatie of koptekst van de pagina worden aangeduid met CSS.




Javascript

JavaScript is een scripttaal om gebruikersinteracties te beoordelen, de inhoud van de website te wijzigen, opnieuw te laden of te genereren.



PHP

We kunnen het niet vaak genoeg zeggen; PHP is geen programmeertaal.
PHP is een serverside scripttaal.
PHP wordt gebruikt om webpagina’s in te delen en tekst weer te geven. Dat is ook precies wat HTML doet, maar PHP heeft een belangrijke toegevoegde waarde.




SQL

Zoals de volledige naam (Structured Query Language) al aangeeft,
is SQL verantwoordelijk voor het bevragen en bewerken van informatie die in een bepaald databasemanagementsysteem is opgeslagen.
